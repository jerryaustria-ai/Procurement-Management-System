# Environment Variable Templates

## Local frontend (`client/.env`)

```env
VITE_API_URL=http://localhost:5001/api
```

## Local backend (`server/.env`)

```env
MONGODB_URI=mongodb+srv://dbUser:dbPassword@cluster0.xxxxx.mongodb.net/januarius_procurement_dev?retryWrites=true&w=majority
JWT_SECRET=local-dev-super-secret-jwt-key-change-me
CLIENT_ORIGIN=http://localhost:5173
PORT=5001
HOST=0.0.0.0
```

## Vercel frontend env

```env
VITE_API_URL=https://your-render-service.onrender.com/api
```

## Render backend env

```env
MONGODB_URI=mongodb+srv://dbUser:dbPassword@cluster0.xxxxx.mongodb.net/januarius_procurement_prod?retryWrites=true&w=majority
JWT_SECRET=replace-this-with-a-long-random-secret-key
CLIENT_ORIGIN=https://your-procurement-app.vercel.app
PORT=10000
HOST=0.0.0.0
```

## Example using your app names

If your frontend URL is `https://januarius-procurement.vercel.app` and your backend URL is `https://januarius-procurement-api.onrender.com`, use:

### Vercel

```env
VITE_API_URL=https://januarius-procurement-api.onrender.com/api
```

### Render

```env
MONGODB_URI=mongodb+srv://dbUser:dbPassword@cluster0.xxxxx.mongodb.net/januarius_procurement_prod?retryWrites=true&w=majority
JWT_SECRET=Januarius-Procurement-2026-Secure-Secret-Key
CLIENT_ORIGIN=https://januarius-procurement.vercel.app
PORT=10000
HOST=0.0.0.0
```
